# Algorithms-for-Massive-Data-Project
The project features the implementation of two main algorithms used to solve the problem of finding frequent itemset, that are A-Priori and PCY (Park-Chen-Yu).
In particular, the project requires to solve the task of finding frequent sets of actors inside the IMDB Movies Dataset, containing the top 1000 movies by IMDB rating. This dataset is downloaded automatically in the notebook using the Kaggle API.
For the purpose of the project, each film has been treated as a basket, while the actors playing in it as its elements. The goal of the project is then to find frequent couples of actors that played together in higly rated movies. 

In this repository one can find:

* The python notebook containing the relevant code
* The report in PDF containing a detailed explanation of the whole workflow and resuming the results project

The notebook is executable directly in Google Colab, by clicking the relative Colab badge.
